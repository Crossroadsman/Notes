Responsive Design
=================

Key concepts are:
- fluid images,
- fluid,
- media queries.

Media Queries
-------------

Syntax:

```CSS
@media not|only mediatype and (media feature) {

  more css goes here {
  
  }
}
```

`not|only` are optional

`mediatype` can be any of:

- all
- print
- screen
- speech

`media feature` goes in parens and looks like: `(min-width: 480px)`
